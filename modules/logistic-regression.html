Stage 02

Why Lines Break 📏

Your first instinct might be to draw a straight line through the data — that's linear regression.

But watch what happens: the line predicts values above 1 and below 0. Those aren't valid probabilities!

ŷ = b₀ + b₁ · Temperature Can give 1.5 or −0.3 … not a probability!

We need a function that squishes any number into the 0–1 range. Enter the sigmoid.

Stage 03

The Sigmoid Function ✨

The sigmoid function takes any score and maps it to a probability between 0 and 1.

p = 1 / (1 + e⁻ᶻ) Large positive z → probability near 1 Large negative z → probability near 0 z = 0 → probability exactly 0.5

Stage 04

Weighted Clues 🧠

Where does z come from? The model assigns a weight to each feature and adds them up:

z = b₀ + b₁·Temp + b₂·Sunny + b₃·Weekend

b₀ = bias · b₁, b₂, b₃ = how much each clue matters

Stage 05

Learning from Mistakes ⚙️

The model starts with initial weights, makes predictions, measures how wrong it was using log loss, then nudges the weights to improve. This is gradient descent.

L = −[ y·log(p) + (1−y)·log(1−p) ] The model is punished more when it's very confident but very wrong.

Stage 06

Drawing the Line 🚦

The sigmoid gives a probability. But we need a decision: Yes or No. We pick a threshold.

If p ≥ threshold → predict Yes (1)
If p < threshold → predict No (0)

Set Threshold

0.50

💡 A medical test might use 0.3 (catch more sick people), while a spam filter might use 0.7 (avoid false alarms).

Stage 08

The ROC Curve 🔔

The ROC curve shows: at every possible threshold, how does recall trade off against false-alarm rate?

Think of a smoke alarm: too sensitive and it goes off when you cook toast; too lenient and it misses an actual fire.

—

📌 Note: Because we only have 6 test samples, this ROC curve looks coarse and jumpy. With a real dataset of hundreds of rows, the curve would be much smoother.

Stage 09

What the Numbers Mean 📎

Each coefficient tells a story:

Positive weight → pushes probability up
Negative weight → pushes probability down
Larger magnitude → stronger influence

We can compute odds ratios: ebᵢ tells you how the odds multiply for each unit increase in that feature.
